# Phone Catalog App


### Build with:

  * **[Angular](https://angular.io/)**
  * **[Express](http://expressjs.com/)**
  * **[Docker](https://www.docker.com/)**


#### Docker:

* Dev project and watch
```bash
$ make dev
```

* Install_front
```bash
$ make install_front
```

* Dev front
```bash
$ make dev_front
```

* Build front
```bash
$ make build_front
```

* Install back
```bash
$ make install_back
```

* Build back
```bash
$ make build_back
```

* Dev back
```bash
$ make dev_back
```

* Up containter
```bash
$ make up
```
